Токен авторизации GitHub

Для автоматической записи измененного файла menu.json в репозиторий требуется GitHub Personal Access Token (PAT) с правами на запись (права repo или доступ к контенту репозитория).

Ваш токен сохраняется локально только в вашем собственном браузере (в localStorage) и передается только напрямую на официальный сервер api.github.com.
